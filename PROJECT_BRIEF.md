# База Знаний Проекта "Open Deck DJ School" (Расширенная Версия)

> Источник истины по проекту. Обновлено: 2026-07-20.

## 1. Философия и Миссия (ДНК Бренда)

**Главный Манифест:** "Open Deck — Open Heart — Open Mind." Операционная система, пронизывающая продукт и коммуникацию.

- **Open Deck (Свободная Дека):** учим не "нажимать кнопки", а "играть" музыкой и через неё — "играть в жизнь". Метафора смелости быть собой, не бояться ошибок, превращать их в свой стиль. В курсе нет "правильных"/"неправильных" решений — есть отражающие внутренний мир. Снимает страх перфекционизма.
- **Open Heart (Открытое Сердце):** музыка — язык эмоций. Учим выражать чувства через звук, создавать связи. Сводим треки по настроению, а не только по BPM. "Читать" энергию танцпола.
- **Open Mind (Открытый Разум):** любопытство, выход за рамки привычного, эклектика, поиск своего звучания на стыке жанров.

**Ключевые Инсайды:**
- "Искусство — это способ быть в согласии с собой и с Вселенной." Диджеинг как активная медитация, состояние "потока", практика осознанности.
- "Творческое увлечение — это прочная точка опоры личности." Ментальный "якорь", самоуважение, инвестиция в ментальное здоровье.
- **Подход:** без снобизма и академической скуки, с лёгкостью. Тон — дружеский, как разговор со старшим товарищем.
- **Цель:** дать инструмент, чтобы человек нашёл опору и "открытое сердце зазвучало громче".

## 2. Целевая Аудитория (Наше "Племя") — 5 психотипов

1. **"Кураторы Вайба"** (18+): душа компании, музыкальные авторитеты. Боль — влияние эфемерно, хотят стать "тем, кто создаёт событие", а не "включает с телефона". Страх — стать предсказуемым.
2. **"Мастера Практик"** (25+, йога/ecstatic dance): музыка — рабочий инструмент. Боль — невозможность управлять энергией группы в реальном времени.
3. **"Искатели Творчества"** (25+, IT/финансы): ищут "творческий побег" от рутины. Боль — "день сурка", страх быть "технарём, а не творцом".
4. **"Будущие Артисты"** (18+): видят карьеру в DJ. Боль — не знают, с чего начать, боятся конкуренции.
5. **"Новое Поколение"** (12-17): маркетинг — родители, ищущие здоровую альтернативу "залипанию" в гаджетах.

### Сценарий рекламного креатива 15 сек
- [1с] Портрет с джогом, скретч со звуком. Аудио: фить-фить пиу.
- [1с] Идёшь по улице с контроллером на плече (Дананг). Голос: "Ты живёшь в Дананге?"
- [3с] Таша на фестивалях. "Хочешь стать диджеем и открыть мир музыки по-новому?"
- [3с] Таша выбирается из проводов. "Но всегда казалось, что это сложно и долго?"
- [5с] Таша обучает людей. "Я создала супер-лёгкий курс по диджеингу! Всего за месяц освоишь технику сведения, запишешь первый микс, почувствуешь вдохновение."
- [2с] Таша в кадр. "Первое занятие бесплатно! Пиши мне в директ!"

**Текст в посте:** Научишься сводить треки. Запишешь свой первый сет. Откроешь мир музыки по-новому. Подробнее по ссылке в шапке профиля.

## 9. Программа курса "Pocket DJ" (структура)

Курс из **10 уроков по 15–20 минут**.

1. **Знакомство:** кто такой диджей, зачем нужен, функции на вечеринке, применение в современном мире. Бэкграунд Таши (FREEDA DJ).
2. **История:** краткая история появления диджеинга, развитие электронной музыки от диско до наших дней.
3. **Направления и оборудование:** коммерческий / ивент / андерграунд / фестивальный / радио / проводник практик; виды оборудования, сферы.
4. **Музыкальная теория:** бит, ритм, тональности, колесо Камелота, дорожки; строение трека (интро, ямы, дропы, аутро).
5. **Интерфейс на телефоне:** обзор программы, контроллер, дека, частоты, плей, фейдеры, фильтр.
6. **Основы сведения:** BPM, темп, микширование в одной тональности/стиле/темпе, кнопка Sync.
7. **Кроссфейдер и темпо:** техника сведения, нюансы темпо (Wide / 6 / 10 / 16).
8. **Лупы и Beat Jump:** простое сведение через петли, прыжки по битам.
9. **Эффекты и фильтр:** что такое эффекты, обзор бесплатной версии, простое применение, пады, фильтр при сведении.
10. **Первый микс:** основы композиции, сводим микс из 4–5 треков разными способами.

**Дополнительные материалы (самостоятельно):**
- Поиск и скачивание треков.
- Использование нейросетей в диджеинге.
- Ресурсы для глубокого изучения стилей музыки.
- Список книг и фильмов о диджеях и электронной музыке.

**Бонусы при покупке курса:**
- Доступ к закрытому Telegram-чату комьюнити (живой эфир раз в 2 недели + регулярные посты).
- Общее хранилище музыки «золотая коллекция» (пополняется учениками).
- База организаторов вечеринок по всему миру (личная + от студентов).
- Дополнительные материалы к каждому занятию.

## 11. База данных пользователей (SQLite)

Реальная БД вместо временного словаря в памяти. Файл `dj_school.db` рядом с бэкендом.

**Таблицы (`backend/db.py`):**
- `users` — user_id, first_name, username, photo_url, created_at, last_seen
- `progress` — user_id, course_id, lesson_id, completed, gp_earned (50 GP/урок), completed_at
- `payments` — user_id, course_id, provider (stars/ton/prodamus), amount, currency, status, raw
- `badges` — user_id, course_id, badge

**Бэкенд (`backend/main.py`) пишет в БД:**
- `/api/lessons` — проверяет `has_paid`, отдаёт completed-уроки
- `POST /api/progress` — сохраняет завершение урока (`complete_lesson` → GP), вызывается из приложения
- `POST /api/grant` + `/webhooks/prodamus` — фиксируют оплату
- `db.init()` вызывается при старте (таблицы создаются сами)

**Приложение (`miniapp/index.html`):** `completeLesson()` шлёт `POST /api/progress` с `X-Init-Data` (tg.initData) → прогресс не теряется при перезапуске.

**Запуск бэкенда (важно!):** только из КОРНЯ проекта, иначе `import backend` падает:
```
cd /c/Users/Admin/dj-school-tma
BOT_TOKEN=... python -m uvicorn backend.main:app --port 8000
```
(НЕ из папки backend/ — будет ModuleNotFoundError: No module named 'backend')
